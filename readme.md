## Описание программы

Данная программа предназначена для работы с телефонным справочником. Версия 1 работает только c json файлами со структурой, указанной ниже:

{'id' : {'name': '*value*', 'phone': [*list_of_values*], 'birthday': '*value*', 'email': [*list_of_values*]}}

Программа позволяет создать новую телефонную книгу, импортировать существующую (строго с указанной структурой), экспортировать созданную.

### **При работе с телефонной книгой вы можете:**
- посмотреть все содержимое телефонной книги
- добавить новый контакт
- найти контакт по любому из полей (id, имя, телефон, дата рождения, email)*
- изменить данные уже сохраненных контактов**
- удалить один или сразу несколько контактов

Далее вы найдете инструкции по использованию программы.

_\* (в версии 1 возможности поиска ограничены: контакт можно найти, если полностью ввести имя/телефон/дату/почту. При поиске по телефону/почте достаточно ввести лишь один из номеров или адресов)_

_\*\* поле "id" менять нельзя_

## Начало работы с программой. Главное меню

Для начала работы необходимо запустить файл "main1.py". Откроется главное меню программы как на скриншоте ниже:

 ![start_screenshot](/images/start.png)

 Команды указаны в меню. Для выбора действия напишите цифру желаемого действия. Если введете некорректное значение, программа откроет главное меню и предложит ввести команду еще раз.

### 1. Создание новой телефонной книги

Ввведите команду "1" для создания новой телефонной книги. 
Программа предложит ввести название. 
Если вы ничего не введете, программа укажет на ошибку и попросит ввести название еще раз.

![button1_test1](/images/test1.1.png)

Если название введено корретно, программа сообщит, что телефонная книга создана и переведет в меню работы с телефонной книгой.

![button1_test2](/images/test1.2.png)
___
### 2. Импорт существующей телефонной книги

Если у вас уже есть телефонная книга, вы можете загрузить ее. Нажмите "2" для импорта. Программа предложит ввести название файла (без расширения!).

Если вы ввели название несуществующего файла, программа сообщит, что база данных не найдена и вернет в главное меню.

![button2_test1](/images/test2.1.png)

Если файл найден, программа сообщит, что телефонная книга импортирована и перейдет в меню работы с телефонной книгой.
__Обратите внимание!__ Название файла необходимо вводить __БЕЗ__ расширения. Расширение .json добавится автоматически. 

![button2_test2](/images/test2.2.png)

__ВАЖНО!__ Если вы ввели корректное название файла, но структура самого файла отличается от требований программы, программа не будет работать корректно.

Если эта команда используется, когда никакая телефонная книга еще не открыта или не создана, программа сообщит об этом и вернется в главное меню. 

![button3_test1](/images/test3.1.png)

## Работа с телефонной книгой

Меню использования телефонной книги открывается автоматически при создании новой или импорте существующей книги, а также при выборе команды "4" в главном меню. 

![menu2_test1](/images/start2.png)
___
### 1. Просмотр всей телефонной книги

Если ваша телефонная книга не большая, можете воспользоваться командой "1" для просмотра всего содержимого справочника и найти необходимый контакт самостоятельно. 

При вызове этой команды показываются только те поля контакта, которые были заполнены при создании. Пустые поля не отражаются.
_____
### 2. Создание нового контакта

Чтобы создать новый контакт, воспользуйтесь командой "2".
Программа предложит ввести имя нового контакта. Имя - обязательный параметр для создания контакта. Если вы не введете имя, процесс создания контакта прервется и придется начать его заново.

![button2_test1](/images/test2.2.1.png)

После имени программа предлагает заполнить следующие поля: номер телефона (для ввода нескольких номеров запишите их через пробел), дата рождения, электронная почта (для ввода нескольких адресов запишите их через пробел). Эти поля не являются обязательными. Если их не заполнить, контакт будет содержать только id и имя.

![button2_test1](/images/test2.2.2.png)

После заполнения всех полей программа сохраняет контакт в текущую телефонную книгу и возвращает меню использования справочника. 

Поле "id" заполняется автоматически. Программа создает id для нового контакта путем прибавления единицы к максимальному id среди существующих контактов.

_____
### 3. Поиск контактов в телефонной книге

Для поиска контактов используется команда "3". После выбора команды программа предлагает выбрать параметр, по которому вы желаете найти контакт.

![button3_test1](/images/test2.3.1.png)

Выберите команду желаемого параметра и введите значение параметра для поиска.

+ поиск по id актуален, если вы точно помните id, так как это прото порядковый номер внесения в справочник, который не несет в себе контактной информации
+ поиск по имени находит контакты: содержащие то имя, которое вы вводите, при этом регистр не имеет значения; версия 1 не поддерживает поиск по части имени
+ поиск по номеру телефона показывает все контакты, у которых есть введенный номер телефона; версия 1 не поддерживает поиск по части номера
+ поиск по дате рождения возвращает контакты, у которых дата рождения та же, которую вы вводите, при этом учтите, что в версии 1 необходимо вводить дату именно в том формате, в котором она записана в справочнике; если в контакте дата - 27-02-1995, для нахождения этого контакта надо ввести именно эти символы. При вводе других форматов (например: 1995-02-27 или дата без года 27-02 и т.д.) поиск не отразит данный контакт
+ поиск по электронной почте аналогичен поиску по номеру телефона; версия 1 не поддерживает поиск по части адреса почты

Также в меню параметров поиска присутствует возможность вернуться в меню работы со справочником с помощью команды "9". Воспользуйтесь ею, если передумали искать контакты.

_____
### 4. Изменение данных контакта

Программа позволяет отредактировать любое поле любого контакта кроме уникального поля "id", которое присваивается контактам автоматически.

Для редактирования контакта сначала необходимо найти контакт. Именно это и предлагает программа, возвращая меню выбора параметра для поиска. Если вы знаете id контакта, рекомендую сразу искать именно по этому параметру, чтобы сразу найти необходимый контакт.

Если вы ищете по другим параметрам и в справочнике нашлось несколько контактов по искомому значению, программа покажет все найденные результаты и предложит ввести id одного контакта, который требуется изменить. Кроме того, вы можете выбрать команду "q" для возврата в менб работы со справочником, если передумали редактировать информацию.

![button4_test1](/images/test2.4.1.png)

Когда результаты поисков сводятся к одному контакту, программа покажет меню полей контакта для выбора поля, значение которого вы будете менять. В этом меню также присутствует команда "9" для возврата в меню работы со справочником, если передумали редактировать информацию.

![button4_test2](/images/test2.4.2.png)

Выберите поле, которое хотите отредактировать и введите новое значение параметра. Значение параметра полностью перезаписывается. Если вы хотите дополнить уже имеющуюся в поле информацию, введите и текущую, и новую информацию.
Например, если имя контакта - Иван и вы хотите, чтобы в поле имя было указано "Иван Иванович", во время редактирования имени напишите новое значение полностью: "Иван Иванович".

После введения нового значения, программа сообщает о сохранении телефонной книги и возвращает в меню работы со справочником.

_____
### 5. Удаление контактов из телефонной книги

Процесс удаления контактов, как и редактирование, начинается с процесса поиска контактов для удаления. Меню поиска также знакомое: возможен поиск по всем полям. 

![button5_test1](/images/test2.5.1.png)

Если поиск вернул два и более результата, программа спросит, хотим ли мы удалить все найденные контакты или только какие-то из них. Нажмите "0" для удаления всех найденных контактов. Нажмите "q" для возврата в меню работы со справочником. Или введите id тех найденных контактов, которые требуется удалить.

Если значение параметра поиска введено некорректно или в справочнике нет таких значений, Программа сообщит, что контакт не найдем и предложит еще раз выбрать параметр.

![button5_test1](/images/test2.5.2.png)


____
### 6. Вовзрат в главное меню

Если вы закончили работу со справочником, просто выберете команду "9" для возврата в главное меню. Сама телефонная книга при этом остается открытой в программе, поэтому вы спокойно можете вернуться к работе без необходимости посторно импортировать json файл своей телефонной книги.

## Планы по доработке программы:
1. Создать возможность общего поиска вне зависимости от поля
2. Создать возможность поиска по части слова/ номера телефона
3. Изменить тип данных в дате рождения на дату
4. Добавить функционал поиска по дате рождения: у кого сегодня день рождения/ у кого день рождения на следующей неделе/ у кого в течение заданного периола времени
5. Создать телеграм-бот с функционалом данной программы
6. Создать возможность работы с разными типами файлов
7. Подключить БД SQLITE3
8. Добавить проверку ввода, чтобы в номера телефонов не вносились буквы.
9. Скорректировать ввод даты при создании контакта, чтобы можно было ввести только дату
